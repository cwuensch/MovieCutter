--------------------------------------------------------------------------------
* Aggregates:
  Blocks 00 - 07: Reserved (64 Sect, 32kB, Size is independend of block size)
         08     : Primary Aggregate Superblock
         09     : Control Page
         10     : IAG
         11 - 14: 32 Aggregate Inodes
                    inode 0 = reserved
                    inode 1 = self
                    inode 2 = block allocation map
                    inode 3 = in-line log if used
                    inode 4 = bad blocks
                    inodes 5 to 15 = reserved
                    inodes 6 to 31 = one inode per fileset

         15     : Secondary Aggregate Superblock
         16 - 19: Block Allocation Map
         20 - 23:  16 Aggregate Inodes
         24     : Control Page
         25     : IAG
         ...    : fsck working space & inline log???

FileSet 0
         60     : Control Page
         61     : IAG
         62     : 32 fileset inodes
